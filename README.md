# Portfolio

## About

Personal portfolio regrouping projects I've worked on.

## Installation

### Step 1 : Clone projet

`git clone https://github.com/Tsuricate/Portfolio.git`

### Step 2 : Install dependencies

`cd YAAC_front`

Using yarn : `yarn`

Using npm : `npm install`

### Step 3 : Run project

Using yarn : `yarn start`

Using npm : `npm start`
